This folder contains the csv version of the periodic table published by pubChem and available at the address [https://pubchem.ncbi.nlm.nih.gov/rest/pug/periodictable/CSV?response_type=save&response_basename=PubChemElements_all](https://pubchem.ncbi.nlm.nih.gov/rest/pug/periodictable/CSV?response_type=save&response_basename=PubChemElements_all).

The script [download_data.py](/data/raw/pubChem/download_data.py) automatically downloads the data.
